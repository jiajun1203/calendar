# calendar
# react-native calendar
  支持单/多选，农历展示，添加标记，首次加载自动添加选择项，左右滑动查看上下月。
 
  选择框大小，样式均可修改。
 
  下次更新按年展示，支持上下滑动展示。

  使用：npm i mrczz-calendar --save

  DEMO:cd到Demo目录下，执行npm install 完成后，直接运行项目
